# Reference Runs

This directory contains real run outputs from the current repo.

The purpose is not to preserve the existing implementation. The purpose is to
show:

- what artifacts were actually written
- how run directories are structured
- what run summaries, logs, and per-course YAML looked like
- which operator workflows were expected to work end to end

Use these as reference examples only.

The rewrite team should preserve only the artifact families and operator
behaviors that remain part of the intended target contract described in
`tmp/handoff/docs/`.
